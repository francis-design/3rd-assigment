GIT STASH

it is a temporary changes one can make to ones working copy so you can work on something else, and then come back and re-apply them later on

git rebase

this is a way you can diverge your work and make commits on two different branches.

GIt merge

git merge command lets you take the independent lines of development created by git branch and integrate them into a single branch,The current branch will be updated to reflect the merge, but the target branch will be completely unaffected.
